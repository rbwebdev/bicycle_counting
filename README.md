#Comptage cyclistes
App ayant pour but de compter les cyclistes pour une association.

### Intallation du projet
```
> git clone git@github.com:rbwebdev/bicycle_counting.git
> cd bicycle_counting
> npm install
> npm run watch
```

### Build prod
```
> npm run build
```
